Pizza delivery

Description

A simple top-down old school gta style Pizza delivery game. This was the first project I made in Unity and was aimed to get to grips with placing objects in the environment and working with colliders / rigid bodies.

Core Mechanics + Game loop

The main game loop was to pick up and then deliver a pizza to a hungover customer. Key mechanics were the ability to pick up a pizza and having to deliver it before picking up the next pizza, a slow down mechanic that impated the car whenever an obstacle was hit as well as a boost function, to speed up the car whenever a boost was driven over.

Additional Features / Improvements

While this was my first attempt at creating a game, I would like to revisit this project at some point and apply additional skills learnt to make this more rounded as a standalone project. Key items to add would be:

  • Scoring system
  • Time limit
  • Random spawning of pizzas + customers
  • A clear UI with key info



Snowboard sidescroller

Description

A simple side scrolling snowbaord game, where players must reach the finish without crashing. The aim of this project was to apply the skills learnt in the delivery game but to work with some other features such as sound effects and gravity.

Core Mechanics + Game loop

The player must guide the snowboarder to the end of the level without crashing. Key mechanics were the ability to rotate the character in the air and to speed up when pressing forward. Only one level was created and this restarts upon completion.

Additional Features / Improvements

Similarly, I would like to revisit this to add some more content to make it a standalone game. Key items to add would be:

  • Add in tricks
  • Additional levels
  • Scoring system
  • A clear UI with key info



Miscellany - a simple quiz Game

Description

As a big fan of quizes, I set up a very simple quiz game. With some features that began to test out some more C# scripting, such as the timer and the progress bar I really enjoyed starting to see how the different components of larger games are added. You can play my quiz game either using the link below or here on simmer.io

Core Mechanics + Game loop

The player must answer a set of quiz questions and is given a score based on performance . There is a time limit of 15 seconds per question, after which the player will not score. The player is presented with their score at the end and can try the quiz again.

Additional Features / Improvements

With this being my first game which has an intro screen and navigation for the user to start / restart the quiz, Key items to add would be:

  • More questions +
  • Randomise questions
  • Add a different style of question
