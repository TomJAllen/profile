FlexBox

A Ruby on Rails Progressive Web Application which helps people make their perfect workout. FlexBox suits all fitness users from beginner gym goers to professional athelets. It provides strucutre to create a workout from our extensive database of exercises, save favourite workouts and see a workout history. The simple format of the app allows users to plan a well rounded warmup, workout or cooldown. Built with an interactive workout feature that guides your through your specified workout, Spotify Integration and a workout history - we have many more features planned!

Each game was developed in the Unity engine and used some simple C# scripts to run

Pizza delivery

A simple top-down old school gta style Pizza delivery game.

Snowboard sidescroller

A simple side scrolling snowbaord game, where players must reach the finish without crashing.
