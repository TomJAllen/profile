Game Development

Gaming has been my passion since receiving my first console at age 6. It has provided me with countless memorable experiences - from conquering Dark Souls bosses to solving challenging Myst puzzles, achieving clutch victories in Warzone/Fortnite, and immersing myself in masterful storylines like Metal Gear Solid and God of War. Gaming's depth and immersion have helped me maintain connections with distant friends and forge new ones.

With the advent of accessible game engines, I'm excited to transform this hobby into a more permanent part of my life. Leveraging my development skills and creative ideas, I've begun exploring game development using Unity. Following GameDev.tv's curriculum, I've created three initial projects to understand how to implement my ideas: a retro-style pizza delivery game, a side-scrolling snowboard adventure, and an engaging quiz game.

Pizza Delivery

Description

A retro-style top-down pizza delivery game inspired by classic GTA. This was my first Unity project, focusing on environment object placement and physics interactions.

Core Mechanics & Game Loop

Players pick up and deliver pizzas to customers while managing time constraints. Key features include:

  • Pizza pickup and delivery system
  • Vehicle collision physics with slowdown effects
  • Speed boost power-ups

Future Enhancements

  • Dynamic scoring system
  • Time-based challenges
  • Procedural pizza and customer spawning
  • Enhanced UI/UX design

Snowboard Side-scroller

Description

A challenging side-scrolling snowboard game where players navigate through obstacles to reach the finish line. This project expanded my skills in sound design and physics implementation.

Core Mechanics & Game Loop

Navigate through challenging terrain while maintaining momentum. Features include:

  • Aerial rotation mechanics
  • Speed control system
  • Dynamic level design

Future Enhancements

  • Trick system implementation
  • Multiple level designs
  • Progressive difficulty scaling
  • Enhanced visual feedback

Miscellany - Quiz Game

Description

A dynamic quiz game showcasing advanced C# scripting and UI implementation. Test your knowledge with timed questions and track your progress.

Play the game: Launch on Simmer.io

Core Mechanics & Game Loop

Challenge yourself with timed questions and track your performance:

  • 15-second time limit per question
  • Score tracking system
  • Progress visualization

Future Enhancements

  • Expanded question database
  • Dynamic question randomization
  • Multiple question formats
  • Category-based gameplay
